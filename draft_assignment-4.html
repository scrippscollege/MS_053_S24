Drawing Machine

Objective

Design and code a drawing machine.

Description

Strictly speaking, modern software is never fully completed. For this assignment you will write the next version of your Face Generator by re-arrranging the code into functions.

Timeline

Part 1 : Due 2/19

  1. Choose an object you have lost in the past that has meaning to you.
  2. Describe your memory of the object using plain language. It is up to you how much detail you impart, but be mindful of your own assumptions about what your partner might know about the object.
  3. Share your description (without any reference images) with your assigned partner over Canvas.

Part 2: Due 2/26

  1. After receiving your partner's description, render their object by drawing on a piece of paper. Once you're satisfied with your design, transfer the drawing into code. Be sure to include the following technical components in your sketch:
    • function
    • passing the parameters

Submission Guidelines
